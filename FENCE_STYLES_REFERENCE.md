## Notes

- **Categories** are case-sensitive - use exact capitalization as shown above
- **Subcategories** are also case-sensitive - use exact names from the fence style lists
- The `active` column: use `1` for active products, `0` for inactive
- You can leave `subcategory` blank if not applicable
- Each fence style has specific subcategories - refer to the "Fence Styles with Available Subcategories" section above
- When importing via CSV, ensure category and subcategory combinations are valid (e.g., "Glass Panels" is valid for "Frameless Pool Fence" but not for "Zeus Steel Fencing")
