2) Implement/fix Equalize @ 50 mm
   Files:
   - shared/calc/equalize.ts (create if missing)
   - server/services/resolve.ts (integration)

   Implement:
   export function equalizePanels({
     targetMm, stepMm=50, maxPanelMm, minPanelMm=300
   }): { widthsMm:number[], error?:string }

   Algorithm:
   - Start with count = ceil(targetMm / maxPanelMm) (at least 1).
   - Loop up to count+20:
       base = floor((targetMm / count) / stepMm) * stepMm
       leftover = targetMm - base*count
       steps = round(leftover / stepMm)
       widths = Array(count).fill(base)
       for i in 0..steps-1: widths[i] += stepMm
       if every width in [min,max] AND sum(widths)==targetMm (±2 mm), success.
       else count++ and retry.
   - If no solution, return error describing infeasible bounds.
