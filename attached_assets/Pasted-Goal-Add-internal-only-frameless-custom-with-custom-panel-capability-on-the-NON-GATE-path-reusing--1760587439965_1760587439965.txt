Goal:
Add internal-only frameless_custom with custom panel capability on the NON-GATE path, reusing the existing equalize & sequencing rules. Do NOT modify the gate flow.

Scope:
- New CustomPanelConfig in compose/lengthAccounting
- Non-gate path supports: fixed custom panel (width/height), optional gaps before/after, optional position
- Equalize the remainder on 50 mm grid (LOCKED_OR_RESIDUAL default)
- Tests + a debug endpoint; no UI yet

    required: boolean;

2) Fixed accounting for custom panel
- In shared/calc/lengthAccounting.ts add:
  export interface CustomPanelFixed {
    customPanelMm: number;          // width
    customGapsMm: number;           // gapBefore + gapAfter
    fixedLeftMm: number;            // for position-based split; for now compute as 0 (Phase 1 keeps simple insertion later)
    fixedRightMm: number;
  }
  export function computeFixedCustomPanel(
    custom: CustomPanelConfig | undefined,
    defaultHeightMm: number
  ): CustomPanelFixed {
    if (!custom?.required) return { customPanelMm:0, customGapsMm:0, fixedLeftMm:0, fixedRightMm:0 };
    const width = Math.max(0, Math.round(custom.panelWidthMm));
    const gaps = Math.max(0, Math.round((custom.gapBeforeMm ?? 0) + (custom.gapAfterMm ?? 0)));
    return { customPanelMm: width, customGapsMm: gaps, fixedLeftMm:0, fixedRightMm:0 };
  }

3) Extend non-gate composer (composeFenceSegments)
- In shared/calc/compose.ts, within the ELSE (non-gate) branch:
  a) Call computeFixedCustomPanel(customPanelConfig, defaultHeight).
  b) Compute F = startGapMm + endGapMm + customPanel.customPanelMm + customPanel.customGapsMm
  c) LOCKED first:
     targetPanels = runLengthMm − F
     Find N via findFeasibleN({ targetMm:targetPanels, minPanelMm, maxPanelMm, stepMm:50 })
     If not feasible and policy==='LOCKED_STRICT' → push error UNREACHABLE and return.
  d) If not feasible and policy LOCKED_OR_RESIDUAL → provisional = runLengthMm − (startGapMm + (customPanel width+gaps)); solve equalize; set residualEndGap = runLength − (startGap + custom + sum(panels)+(N−1)*between); variance = residual − requested; ensure residual ≥ 0, else increase N while ≤ Nmax or return UNREACHABLE.
  e) Build segments:
     - start gap
     - “left panels” (if placing custom at some position) — PHASE 1: place custom centrally if position omitted, or compute leftShare = round((position || 0.5) * (sum(panels) + (N−1)*between)) and split panel array at nearest between boundary.
     - ‘panel’ segments for left set, separated by betweenGap
     - custom panel (kind:'panel', widthMm:customPanelMm, heightMm:custom.panelHeightMm ?? defaultHeightMm, note:'custom')
     - custom.gapAfterMm as a gap segment if >0
     - right panels separated by betweenGap
     - end gap (requested or residual per policy)
  f) Invariants:
     - No gate segments allowed on non-gate path
     - All panel widths %50==0; length conserved (±1 mm with gate=false → use ±2 mm as current code)
  g) Trace:
     - {source:'custom-panel', width:customPanelMm, gaps:customGapsMm, position:custom.position ?? 0.5}
     - {source:'equalize', step:50, count:N, panelsMm:sum(equalized)}
     - {source:'end-gap-policy', policy, residualUsed, varianceEndGapMm}
